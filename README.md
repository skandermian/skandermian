## 📌 Featured Projects

### 🏦 Credit Risk Prediction using Machine Learning and Neural Networks

Final-year dissertation project focused on predicting credit risk using machine learning on the Home Credit Default Risk dataset, a realistic public lending dataset containing 307,511 loan applications, 122 raw features, and an approximately 8% default rate.

**Highlights:**

* Built a reproducible machine learning pipeline for credit-risk prediction
* Compared logistic regression, XGBoost, and a compact multilayer perceptron model
* Applied missing-value handling, categorical encoding, feature engineering, stratified splitting, class-imbalance handling, and hyperparameter tuning
* Evaluated models using accuracy, balanced accuracy, ROC-AUC, PR-AUC, precision, recall, F1-score, Brier score, and calibration analysis
* Used SHAP analysis to explore model explainability
* Found that XGBoost achieved the strongest ranking performance, with ROC-AUC 0.760 and PR-AUC 0.249
* Discussed responsible AI concerns including calibration, fairness, interpretability, and governance in automated lending

**Tech Stack:** Python · pandas · NumPy · scikit-learn · XGBoost · SHAP · matplotlib · seaborn · Google Colab

### 🫁 Pneumonia Detection using Machine Learning

Academic machine learning project exploring whether supervised learning could support faster and more reliable pneumonia diagnosis using both numerical chest X-ray measurements and raw chest X-ray image classification.

**Highlights:**

* Built an end-to-end machine learning pipeline for tabular pneumonia data
* Applied data cleaning, feature preparation, stratified train/test splitting, model comparison, hyperparameter tuning, and ensemble methods
* Compared multiple supervised classifiers using balanced accuracy, F1-macro, and per-class recall
* Extended the project by implementing a compact PyTorch CNN on raw chest X-ray images
* Compared raw-image deep learning against selected tabular models
* Considered medical AI risks including dataset bias, explainability, clinical validation, and deployment limitations
* Concluded that CNN-based image classification showed useful potential, but introduced trade-offs around error balance, training cost, explainability, and clinical deployment risk

**Tech Stack:** Python · pandas · NumPy · scikit-learn · PyTorch · torchvision · matplotlib · seaborn · Google Colab
